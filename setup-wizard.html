🚀 NCBUS 快速配置向导

只需3步，完成云端同步配置

1 配置 GitHub Token
💡 什么是 Token？ Token 是访问 GitHub 的密钥，用于将您的数据保存到 GitHub Gist。
📝 如何获取 Token？
  1. 打开 GitHub Token 设置
  2. 点击 "Generate new token (classic)"
  3. 勾选 gist 权限
  4. 点击 "Generate token"
  5. 复制生成的 Token 粘贴到上方
2 测试云端同步
✓ 配置完成！
🎉 云端同步已配置成功！

您可以直接打开 V2.14.html 使用云端同步功能了。
